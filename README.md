## Usage

1. **Open Command Palette**: Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac) to open the Command Palette.
2. **Run the Command**: Type "AI open" in the Command Palette and select the AI shortcut from the list (see screenshots above).
